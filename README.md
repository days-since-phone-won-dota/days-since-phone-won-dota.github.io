# Dota 2 Account Statistics Page

Welcome to the Dota 2 Account Statistics Page project! This project provides a humorous and engaging way to view your Dota 2 account statistics, win rates with friends, recommended hero bans, testimonials, and entertaining Twitch clips. It also features a tipping system and playful UI elements inspired by Dota 2.

## Project Structure

The project is organized as follows:

```
days-since-phone-won-dota
├── assets
│   ├── audio
│   │   ├── clips.js               # Handles audio clips
│   ├── cursors
│   │   ├── paw-default.png
│   │   └── paw-pointer.png
│   ├── images
│   └── video
│       └── clips.js               # Handles Twitch clips section
├── css
│   └── styles.css                 # Styles for the application
├── js
│   ├── main.js                    # Main JavaScript entry point
│   ├── statistics.js              # Functions for game statistics
│   └── testimonials.js            # Manages testimonials section
├── index.html                     # Main entry point of the application
└── README.md                      # Project documentation
```

## Features

- **Game Statistics**: View your Dota 2 game statistics in a user-friendly and humorous format, split into recent and overall stats.
- **Win Rate with Friends**: Analyze your win rates with friends, complete with playful commentary.
- **Recommended Hero Bans**: Get funny suggestions for hero bans based on your gameplay.
- **Testimonials**: Read light-hearted testimonials from fellow players.
- **Twitch Clips**: Watch entertaining Twitch clips related to Dota 2.
- **Tipping System**: Send virtual tips with a floating button, complete with Dota 2-style shard animations and sound effects.
- **Party Invitation Popup**: Experience a Dota 2-style party invite when you load the page.
- **Dark/Light Mode**: Toggle between dark and light themes for your preferred viewing experience.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the application.
3. Ensure you have an internet connection to fetch Twitch clips and any external resources.
4. For full functionality, keep all assets in their respective folders as shown above.
